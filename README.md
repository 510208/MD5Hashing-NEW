# MD5Hashing-NEW

<center>中文版 ／ <a href=MD5Hasher_NewEN.md>英文版</a></center>

## 說明

此工具是基於Microsoft VisualBasic.NET所開發，允許使用者利用內建功能檢查檔案完整性，並且盡量縮減主程式體積，讓使用者有全新的安全性體驗！

## 特點

- 儲存空間使用最輕量化
- 改善[舊版本](https://github.com/510208/md5hashing-new)
- 增加內建更新檢查模式（暫為測試版功能）
- 介面比起舊版來得更簡易
- 驗證完整性來源視窗更新自動檢查語法模式

## 已知問題

* 更新檢查有時會造成無法建立SSL錯誤
    * 建議自行檢查更新

安裝 Visual .NET 4.5

請按照以下步驟安裝 Visual .NET 4.5 環境：

  1. 前往 Microsoft 官方網站下載 Visual .NET 4.5：https://microsoft.com/visual-dotnet-4.5
  2. 依照下載頁面上的指示進行安裝。

安裝 Python

請按照以下步驟安裝 Python：

  1. 前往 Python 官方網站下載 Python 安裝程式：https://python.org/downloads
  2. 選擇適合您作業系統的版本並下載安裝程式。
  3. 執行安裝程式，按照指示完成安裝。

安裝 requests 模組

使用以下程式碼在命令提示字元或終端機中安裝 `requests` 模組：

            pip install requests
